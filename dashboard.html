🚀 Quick Start

python attendance_system.py
Start the camera-based attendance system using blink liveness and dataset-gallery matching.
python admin_panel.py
Open the terminal-based administrative panel for student management, model retraining, and report export.

📋 System Workflow

1. The teacher creates or updates the class schedule.
2. The camera module becomes active only during the scheduled attendance window.
3. The student faces the camera and completes blink-based liveness verification.
4. The captured face is evaluated using CNN anti-spoofing analysis and dataset-gallery matching.
5. Attendance is saved automatically after successful authentication.
