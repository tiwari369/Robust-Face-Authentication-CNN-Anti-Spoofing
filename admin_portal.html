👤 Add Student

Copy the command below and paste it into the terminal
1

Open the project folder in the terminal. On Windows, open the folder, press Shift + Right Click, and choose "Open PowerShell window here" or open it directly in VS Code Terminal.

2

Run this command:

python admin_panel.py
3

Enter the password → Press option 1 → Type the student name → Capture three webcam angles by pressing SPACE

4

After capturing photos, model retraining is required. Run option 2 as well.

💡 Keep the webcam window active, align the face inside the detection box, blink once, and press SPACE to force a capture attempt if automatic capture is delayed.

🧠 Retrain Model

This step is required after adding or deleting students
1

Run this command in the terminal:

python admin_panel.py
2

Press option 2 → The model will train automatically

Training may take 30 to 90 seconds depending on the number of students

3

When the message "✅ Model saved!" appears, training is complete. Start the attendance system after that.

⚠️ A new student will not be recognized until the model is retrained.

📊 Export Excel

CSV files from the attendance folder → formatted .xlsx files
1

Install openpyxl first if required:

pip install openpyxl
2

Run the admin panel:

python admin_panel.py
3

Choose option 3 → Select a file number or press A to export all files

The Excel file will be saved inside the attendance/ folder.

📋 Student List View

All registered students with available image counts
1

Run this in the terminal:

python admin_panel.py
2

Press option 4 → The student list will appear with photo counts

🗑️ Delete Student

⚠️ This action is permanent and cannot be undone
1

Run this in the terminal:

python admin_panel.py
2

Choose option 5 → Select the number → Type "yes" to confirm

3

After deletion, retrain the model using option 2

⚠️ After deleting a student, all related photos will also be removed.

📷 Start Attendance System

Time-locked face recognition camera
1

Set the schedule from the teacher portal, or use the preconfigured demo schedule

2

Run this in the terminal:

python attendance_system.py
3

The student stands in front of the camera, completes blink liveness verification, the captured face is matched with enrolled photos, and attendance is saved.

If a phone or screen is shown, the system can flag a spoof warning.

4

Press Q in the camera window to close it
